## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **CMS:** MDX for content
- **Hosting:** Vercel (initially)
- **Analytics:** Plausible (privacy-first)
